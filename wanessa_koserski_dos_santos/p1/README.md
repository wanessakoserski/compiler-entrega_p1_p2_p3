### Compilar programa
```
gcc -o compiler compiler.c
./compiler programa.lpn assembly.asm
```

### Criar binário 
```
gcc -o assembler assembler.c
./assembler assembly.asm binary.bin
```

### Executar binário 
```
gcc -o neader neader.c 
./neader binary.bin
```
